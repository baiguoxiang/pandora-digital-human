# 潘多拉数字人网站 - GitHub Pages 部署指南

## 第 1 步：启用 GitHub Pages
1. 在刚刚打开的页面中，找到 **Source** 下拉菜单
2. 选择 **GitHub Actions**
3. 点击 **Save**

## 第 2 步：等待部署完成
1. 回到仓库页面：https://github.com/baiguoxiang/pandora-ai-platform
2. 点击 **Actions** 标签
3. 你会看到正在运行的部署工作流
4. 等待 1-2 分钟，直到工作流完成（显示绿色勾勾）

## 第 3 步：访问网站
部署完成后，你的网站将在以下地址可用：
https://baiguoxiang.github.io/pandora-ai-platform

## 注意事项
- 首次访问可能需要几分钟才能生效
- 如果以后修改了代码，GitHub Actions 会自动重新部署
- 你可以绑定自己的域名（在 Settings → Pages 中设置 Custom domain）
